# 坦克大战游戏 (中文版)

一个使用Python和Pygame开发的经典坦克大战游戏，完全中文化界面。

## 游戏特色

- 🎮 经典坦克大战玩法
- 🚀 玩家坦克速度是敌方坦克的10倍
- 💥 玩家子弹速度是敌方子弹的10倍
- 🛡️ 玩家坦克有10点生命值，敌方坦克1点生命值
- 🏗️ 随机生成的墙体掩护
- 📊 3个关卡，每关需要击毁10个敌方坦克
- ⏰ 每关开始前有5秒倒计时
- 🎯 智能敌方AI

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行游戏

```bash
python tank_battle.py
```

## 游戏控制

- **移动**: 方向键 或 WASD
- **射击**: 空格键
- **重新开始**: R键（游戏结束后）

## 游戏规则

1. 玩家坦克（绿色）需要击毁所有敌方坦克（红色）
2. 每关需要击毁10个敌方坦克才能进入下一关
3. 玩家坦克被击中10次后游戏结束
4. 敌方坦克被击中1次即爆炸
5. 墙体可以阻挡子弹和坦克移动
6. 通过所有3关后显示"胜利"
